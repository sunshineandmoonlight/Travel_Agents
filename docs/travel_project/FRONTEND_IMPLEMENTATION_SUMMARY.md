### 2. 认证后端API

**文件**: `app/routers/auth.py`

实现的端点：
- `POST /api/travel/auth/register` - 用户注册
- `POST /api/travel/auth/login` - 用户登录
- `POST /api/travel/auth/refresh` - 刷新令牌
- `POST /api/travel/auth/logout` - 用户登出
- `GET /api/travel/auth/me` - 获取当前用户信息
- `PUT /api/travel/auth/me` - 更新用户信息
- `POST /api/travel/auth/change-password` - 修改密码
- `DELETE /api/travel/auth/me` - 删除账户

特性：
- JWT令牌认证
- 密码加密（bcrypt）
- 自动降级到模拟数据
- 完整的错误处理

## 设计系统优化 (2026-03-10)

基于 **ui-ux-pro-max** 设计技能系统，对核心页面进行了专业级优化：

### 应用的设计规范

#### 1. 配色系统
- **主色调**: 天空蓝 `#0EA5E9` - 传递信任与探索感
- **辅助色**: 浅蓝 `#38BDF8` - 渐变过渡
- **CTA色**: 探险橙 `#F97316` - 行动号召按钮
- **背景色**: 天空渐变 `#F0F9FF` → `#E0F2FE`
- **文字色**: 深海蓝 `#0C4A6E` - 更好的对比度
- **辅助文字**: `#64748B` - 柔和易读

#### 2. 字体系统
- **主字体**: DM Sans (Google Fonts)
- **字重**: 400, 500, 600, 700
- **字间距**: -0.02em (标题) - 提升可读性

#### 3. Soft UI Evolution 风格
- 演进式软 UI，增强对比度
- 微妙的深度层次
- 现代化美学
- WCAG AA+ 无障碍标准

#### 4. Bento Grid 布局
- Apple 风格模块化卡片
- 16-24px 圆角
- 软色背景 `#F8FAFC`
- 清晰的视觉层次

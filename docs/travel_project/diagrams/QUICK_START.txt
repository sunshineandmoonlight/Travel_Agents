📋 使用步骤：
-----------
1. 访问 https://app.diagrams.net
2. 点击 "File" → "Open from" → "Device"
3. 选择对应的 .drawio 文件
4. 或者直接创建新图表，使用下面的元素参考

🎨 推荐配色方案：
-----------
分析层 (Group A):
- 填充色: #f8cecc (浅粉红)
- 边框色: #b85450 (深粉红)

设计层 (Group B):
- 填充色: #d5e8d4 (浅绿)
- 边框色: #82b366 (深绿)

生成层 (Group C):
- 填充色: #dae8fc (浅蓝)
- 边框色: #6c8ebf (深蓝)

API层:
- 填充色: #ffe6cc (浅橙)
- 边框色: #d79b00 (深橙)

数据存储:
- MongoDB: #e3e3e3 (浅灰)
- Redis: #e3e3e3 (浅灰)

外部API:
- 填充色: #fff2cc (浅黄)
- 边框色: #d6b656 (深黄)

📐 推荐形状：
-----------
- 智能体/组件: 圆角矩形 (Rounded Rectangle)
- 数据库: 圆柱形 (Cylinder)
- 用户输入: 胶囊形 (Capsule)
- 流程节点: 矩形 (Rectangle)
- 容器/分组: 泳道 (Swimlane)

🔗 连接线样式：
-----------
实线 (________): 主要数据流
虚线 (-------): 辅助数据流 / 依赖关系

线条粗细:
- 主要流程: 2-3px
- 次要流程: 1-2px
- 依赖关系: 1px

✨ 快速创建提示：
-----------
1. 先创建容器/分组 (使用 Swimlane)
2. 在容器内添加组件 (使用 Rounded Rectangle)
3. 连接组件 (使用 Connector 工具)
4. 调整颜色和样式 (使用 Format Panel)
5. 添加文字说明 (使用 Text 工具)

📝 文字说明建议：
-----------
- 智能体: 使用 "名称 + 功能" 格式
  例: "需求分析 Agent&#xa;UserRequirementAnalyst"
- 连接线: 添加简要说明
  例: "用户画像" / "候选目的地"
- 容器: 标注层级或分组
  例: "Group A - 分析层"

🔧 高级技巧：
-----------
1. 对齐和分布: 选中多个元素 → Arrange → Align/Distribute
2. 复制样式: 选中参考元素 → Format → Copy Style → 粘贴到其他元素
3. 快捷键:
   - Ctrl+D: 复制元素
   - Ctrl+G: 组合
   - Ctrl+Shift+G: 取消组合
   - F2: 编辑文字
4. 导出: File → Export as → PNG/SVG/PDF

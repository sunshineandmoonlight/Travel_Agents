### 🌅 上午：故宫博物院 (09:00-12:30)

**入口**: 午门 (南门)
**建议到达时间**: 08:45 (避开旅行团高峰)

#### 游览路线设计
```
午门 → 太和殿 → 中和殿 → 保和殿
→ 乾清宫 → 坤宁宫 → 御花园 → 神武门(北门)
```

#### 重点景点详解

**1. 太和殿 (金銮殿)**
- *简介*: 明清两代皇帝举行大典的地方，中国现存最大的木结构宫殿建筑
- *看点*:
  - 殿内藻井上的蟠龙衔珠
  - 殿前铜鹤与铜龟 (寓意长寿)
  - 台基上的排水龙头 (千龙吐水奇观，雨时可见)
- *拍照点*: 太和殿广场正面全景，避免逆光拍摄
- *游览时间*: 30分钟

**2. 乾清宫**
- *简介*: 明清两代皇帝的寝宫及处理政务之地
- *看点*:
  - "正大光明"匾额 (雍正皇帝手书)
  - 皇帝宝座及屏风
  - 殿内保存的清宫旧藏
- *历史故事*: 顺治皇帝在此亲政、康熙皇帝在此听政

**3. 御花园**
- *简介*: 皇家园林，皇帝休憩之处
- *特色*: 古树参天、假山叠石、亭台楼阁
- *必看*: 堆秀山 (可登高俯瞰故宫全景)

#### 💡 游览贴士
- 租用电子讲解器: ¥20/台 (午门入口处)
- 推荐聘请专业讲解: ¥200/2小时 (需提前1天预约)
- 穿舒适的鞋子，全程步行约3公里
- 午门入口存包处可寄存行李: ¥10/件

#### 🎫 门票信息
- 旺季门票: ¥60 (4-10月)
- 淡季门票: ¥40 (11-3月)
- 购票方式: 微信公众号"故宫博物院"提前7天预约

### 🏛️ 下午: 天坛公园 (14:30-17:00)

**交通**: 故宫神武门出 → 地铁8号线 (什刹海站) → 换乘5号线 → 天坛东门站
**行程时间**: 约40分钟
**交通费用**: ¥4

#### 景点详解

**1. 祈年殿**
- *简介*: 明清两代皇帝祭天的核心建筑，中国古建筑的巅峰之作
- *建筑奇迹*: 全木结构，无一钉一铆，仅靠榫卯结构屹立600年
- *色彩寓意*:
  - 蓝色屋顶: 蓝天
  - 金色宝顶: 皇权
  - 红色柱子: 吉祥
- *摄影技巧*: 广角拍摄全景，仰拍显气势

**2. 皇穹宇 & 回音壁**
- *简介*: 祭天时存放神位之所
- *声学奇迹*:
  - 回音壁: 两人分站东西对话，声音清晰可闻
  - 三音石: 站在第一石拍手听一次回声，第二石听两次，第三石听三次
- *体验建议*: 人少时效果更佳，避开下午3-4点旅行团高峰

**3. 圜丘坛**
- *简介*: 冬至祭天之处
- *设计玄机*:
  - 台面石板: 中心石是9的倍数，寓意九五之尊
  - 台阶: 9级，阳数之极
  - 栏板: 360块，象征周天

#### 🎫 门票信息
- 联票 (含祈年殿、圜丘坛): ¥34
- 学生票: ¥17

**历史**: 始创于1864年，中华老字号
**预订**: 电话 010-65253871 (建议提前1天订位)

| 鸭种 | 价格 | 特点 | 推荐指数 |

1. 取一张小饼
2. 放上鸭皮蘸甜面酱
3. 加上葱丝、黄瓜条
4. 卷起送入口中
5. 感受皮脆肉嫩的口感！

**方案二: 包车/拼车 (舒适便捷)**
- 价格: ¥80-120/人 (4人拼车)
- 用时: 约1.5小时
- 推荐平台: 携程、去哪儿网提前预订

#### 慕田峪长城简介

| 项目 | 详情 |
|------|------|
| *历史* | 明洪武元年(1368)始建，隆庆三年(1569)谭纶、戚继光主持修建 |
| *特点* | 保存完好、游客较少、景色壮丽 |
| *最佳拍摄点* | 14号敌楼 (《非诚勿扰2》取景地) |
| *全长* | 5400米 (开放段) |
| *海拔* | 最高处敌楼海拔1003米 |

#### 登城路线推荐

**路线A: 经典环线 (适合体力中等者)**
```
检票口 → 摆渡车上山 → 6号敌楼
→ 14号敌楼 (拍照打卡)
→ 20号敌楼 (最高点)
→ 14号敌楼 → 滑道下山
```
- 用时: 3-4小时
- 难度: ⭐⭐⭐
- 精彩程度: ⭐⭐⭐⭐⭐

**路线B: 深度徒步 (适合体力充沛者)**
```
检票口 → 步行上山 → 1号敌楼
→ 20号敌楼 → 23号敌楼 (野长城)
→ 原路返回 → 步行下山
```
- 用时: 5-6小时
- 难度: ⭐⭐⭐⭐⭐
- 体验: 人少景美，适合摄影

#### 📸 最佳拍照点
1. **14号敌楼**: 《非诚勿扰2》葛优舒淇表白地，长城与群山交相辉映
2. **20号敌楼**: 俯瞰群山，景色最为壮观
3. **烽火台**: 夕阳西下时分拍摄最佳

### 🏛️ 下午: 定陵地宫 (15:00-17:30)

**交通**: 慕田峪 → H24路 → 916路快车 → 昌平县城 → 314路 → 定陵
**行程时间**: 约1.5小时

#### 定陵简介
- *皇帝*: 明神宗朱翊钧 (万历皇帝)
- *皇后*: 孝端皇后、孝靖皇后
- *发掘时间*: 1956-1959年
- *地位*: 第一座按考古计划发掘的帝王陵寝

#### 地宫探秘

**1. 地宫入口**
- 深度: 地下27米
- 建筑面积: 1195平方米
- 门票: ¥45

**2. 宝城宝顶**
- *外观*: 圆形夯土堆，直径约200米
- *含义*: 象征天圆地方
- *攀登*: 可登上宝顶俯瞰地宫全景

**3. 地宫内部**
- **配殿**: 陈列出土文物
  - 金丝翼善冠 (万历皇帝皇冠)
  - 凤冠 (皇后礼冠)
  - 百子衣 (绣有100个童子的衣服)
  - 出土瓷器、金银器

- **玄宫**: 埋葬帝后棺椁之处
  - 前、中、后三殿
  - 后殿置三具红漆棺椁
  *注: 出土文物现已收藏于定陵博物馆*

#### 💡 参观贴士
- 地宫内较阴凉，建议带薄外套
- 参观时间: 1.5小时足够
- 可请讲解: ¥100/团 (现场拼团)

### 🍜 晚餐: 昌平特色 - 阳坊涮肉 (18:30-20:00)

**餐厅**: 阳坊涮肉 (昌平店)
**地址**: 昌平区昌平镇鼓楼南街
**特色**: 正宗老北京涮羊肉

#### 涮肉吃法指南

**第一步: 调小料**
```
推荐搭配:
- 芝麻酱 (基础)
- 腐乳汁 (增香)
- 韭花酱 (提鲜)
- 辣椒油 (可选)
- 芝麻 + 香菜 (点缀)
```

**第二步: 选羊肉**
| 部位 | 特点 | 推荐指数 |
|------|------|----------|
| 上脑 | 肥瘦相间，肉质鲜嫩 | ⭐⭐⭐⭐⭐ |
| 手切羊肉 | 传统切法，口感好 | ⭐⭐⭐⭐ |
| 黄瓜条 | 纯瘦肉，久涮不老 | ⭐⭐⭐⭐ |

**第三步: 涮烫要诀**
- 水开后转小火 (保持微沸)
- 肉片涮10-15秒变色即可
- 不可久涮 (肉质变老)
- 蘸料后及时食用

#### 推荐搭配
- 主食: 烧饼、芝麻酱糖饼
- 配菜: 大白菜、粉丝、冻豆腐
- 饮品: 豆汁儿 (需适应)、北冰洋汽水

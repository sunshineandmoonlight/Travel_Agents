# 目的地情报智能体设计方案

## 功能概述

在旅行规划系统中添加一个**目的地情报智能体**，实时获取目的地的：
- 新闻资讯
- 风险评估
- 节日活动
- 文化体验
- 风土人情

为用户提供更全面的旅行决策支持。

## 子Agent详细设计

### 1. 新闻搜索子Agent

**职责**：搜索目的地的最新新闻

**数据源**：
- 百度新闻API
- Google News API
- Twitter/X搜索
- 当地新闻网站

**搜索关键词**：
- "{destination} 旅游"
- "{destination} 安全"
- "{destination} 政策"
- "{destination} 新闻"

**输出内容**：
```json
{
  "news": [
    {
      "title": "杭州发布新的旅游优惠政策",
      "source": "杭州日报",
      "published_at": "2026-03-10",
      "url": "https://...",
      "summary": "杭州推出新的旅游优惠政策...",
      "sentiment": "positive"  // positive/neutral/negative
    }
  ]
}
```

---

### 2. 风险评估子Agent

**职责**：分析新闻中的风险因素

**风险类型**：

| 风险类别 | 风险等级 | 说明 |
|---------|---------|------|
| **政治风险** | 🔴 严重 | 政治动荡、政策变化、外交关系 |
| **安全风险** | 🔴 严重 | 战争、恐怖袭击、犯罪率上升 |
| **健康风险** | 🟠 中等 | 疫病爆发、食品安全 |
| **自然灾害** | 🟠 中等 | 地震、台风、洪水 |
| **社会事件** | 🟡 较低 | 罢工、示威、大型活动 |
| **经济风险** | 🟡 较低 | 货币贬值、通胀 |

**风险等级**：
- 🔴 **高风险** - 不建议前往
- 🟠 **中风险** - 谨慎前往，需注意
- 🟡 **低风险** - 正常前往
- 🟢 **安全** - 可以放心前往

**输出内容**：
```json
{
  "overall_risk": "medium",
  "risk_level": 2,  // 1-5, 5最高
  "risk_factors": [
    {
      "type": "natural_disaster",
      "level": "medium",
      "description": "近期可能有台风影响",
      "advice": "关注天气预报，做好防护准备"
    }
  ],
  "recommendation": "谨慎前往，建议购买旅游保险"
}
```

### 3. 活动推荐子Agent

**职责**：搜索当地的节庆、活动、展览

**活动类型**：
- 🎊 传统节日（春节、中秋、泼水节等）
- 🎭 文化活动（展览、演出、艺术节）
- 🏃 体育赛事（马拉松、足球赛）
- 🎪 庙会集市（周末市集、夜市）
- 🎆 庆典活动（烟花大会、灯光秀）

**输出内容**：

}
```

## 集成到旅行规划流程

### 现有流程
```
用户输入 → 目的地分类 → 数据收集 → 景点分析 → 行程规划 → 预算分析 → 完成攻略
```

### 新流程
```
用户输入 → 目的地分类 → 数据收集 → 景点分析 → [目的地情报分析] → 行程规划 → 预算分析 → 完成攻略
                                                                      ↓
                                                    在攻略中添加情报建议
```

async def assess_destination_risk(destination: str, news: list):
    """评估目的地风险"""
    risk_score = 0
    risk_factors = []

    for article in news:
        # 使用LLM分析新闻内容
        analysis = await llm.analyze_risk(article["content"])
        if analysis["has_risk"]:
            risk_score += analysis["severity"]
            risk_factors.append(analysis)

    return determine_risk_level(risk_score)
```
